# Network Security

## Overview
This directory contains tools and techniques for network-level security assessment, reconnaissance, and analysis.

## Contents

### Port Scanning & Service Enumeration
- Nmap Scripts: Advanced port scanning techniques
- Service Detection: Identifying running services and versions
- Banner Grabbing: Extracting service information

### Network Reconnaissance
- DNS Enumeration: DNS query tools and scripts
- WHOIS Lookup: Domain information gathering
- Network Mapping: Topology and architecture discovery

### Protocol Analysis
- Packet Capture: Using tcpdump and Wireshark
- Protocol Dissection: Deep packet inspection
- Traffic Analysis: Network flow analysis

## Tools Used
- Nmap
- Metasploit
- Wireshark
- Burp Suite
- Custom Python scripts

## Getting Started
cd 01_Network-Security
python script-name.py

## Warning
Use these tools only on systems you own or have explicit permission to test!
